# 📊 Sales Summary Dashboard (Streamlit App)

## 🚀 Overview

This is a simple interactive Streamlit application designed to present sales data in a clean and user-friendly way. Instead of sharing static files like CSVs, this app allows users to explore and filter the data dynamically.

---

## ✨ Features

* Displays a **sales dataset**
* Allows filtering by **category** (Electronics / Clothing) using a sidebar dropdown
* Shows:

  * 📋 **Filtered table**
  * 📈 **Sales trend chart**
* Maintains a clean interface using a **sidebar filter**

---

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit

---

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install streamlit pandas
   ```

2. Run the app:

   ```bash
   streamlit run sales_app.py
   ```

---

## 📌 Use Case

This app is ideal for sharing quick insights with non-technical stakeholders, allowing them to interact with the data without needing technical tools.
